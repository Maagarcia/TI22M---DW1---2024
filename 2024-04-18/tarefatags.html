samp

o samp representa um tipo de linguagem que o proprio computador tem com o usuario, ou seja, sua fonte, espaçamento e weight serão providas da propria máquina, como se fossem saidas do computador

kbd

e por fim temo a tag kbd, essa tag serve para definir uma entrada do teclado, por exemplo quando voce manda alguem digitar certa tecla

  • Clique ctrl + x para recortar elementos grifados

  • Clique ctrl + shift + setas para grifar elementos

  • Clique ctrl + shift + p para abrir o menu de codigos, digite wrap para poder modificar o que está grifado e em seguida a tag que voce desejar inserir
